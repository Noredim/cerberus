# PROJECT FRONTEND

Frontend principal:

apps/web/src

O frontend do Cerberus está em produção.

Estrutura principal:

pages/
components/
layouts/
hooks/
services/

Objetivo:

Este documento orienta a IA sobre a organização do frontend.

Fluxo recomendado:

1. Ler este índice.
2. Identificar a tela ou componente envolvido.
3. Ler apenas os arquivos necessários.
4. Propor um plano.
5. Implementar apenas após aprovação.

Regras:

* Nunca fazer busca recursiva em todo o frontend.
* Nunca alterar componentes não relacionados.
* Nunca refatorar telas inteiras sem necessidade.
* Sempre preservar a experiência atual do usuário.
* Ler apenas os arquivos envolvidos na tarefa.
