Objetivo:

Este documento orienta a IA sobre a organização do frontend.

Fluxo recomendado:

1. Ler este índice.
2. Identificar o módulo envolvido.
3. Ler apenas os componentes necessários.
4. Identificar os arquivos impactados.
5. Criar um plano.
6. Implementar apenas após aprovação.

Regras:

* Nunca fazer busca recursiva em todo o frontend.
* Nunca alterar módulos não relacionados.
* Nunca refatorar componentes sem necessidade.
* Sempre preservar a experiência do usuário.
* Ler apenas os arquivos envolvidos na tarefa.

Se um módulo não existir, responder:

"NÃO ENCONTRADO."
